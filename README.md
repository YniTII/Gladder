# 🔌 Ladder Simulator - Игровая Симуляция Релейной Логики

Интерактивный веб-симулятор релейных логических схем (Ladder Diagram, IEC 61131-3) с игровой механикой конвейера.

## 🚀 Быстрый старт

### Локальный запуск

Просто откройте файл `index.html` в любом современном браузере:
```bash
# macOS
open index.html

# Linux
xdg-open index.html

# Windows
start index.html
```

Или используйте локальный сервер:
```bash
# Python 3
python -m http.server 8000

# Node.js (если установлен http-server)
npx http-server -p 8000
```

## 📖 Инструкция по использованию

### Основы работы с LD-редактором

1. **Добавление элементов**: Кликните по элементу в палитре → он добавится на первую ступень
2. **Настройка элемента**: Двойной клик по элементу открывает окно конфигурации
3. **Удаление элемента**: Ctrl+Клик по элементу
4. **Новая ступень**: Клик по пустой области редактора
5. **Ручное управление**: Клик по контакту с типом "Ручное управление" переключает его состояние

### Типы элементов

| Элемент | Описание | Поведение |
|---------|----------|-----------|
| НО Контакт | Нормально открытый контакт | Пропускает ток, когда замкнут (state=true) |
| НЗ Контакт | Нормально закрытый контакт | Пропускает ток, когда разомкнут (state=false) |
| Катушка | Выходной элемент | Активируется при наличии тока, управляет выходами |
| Ветка ↑ | Начало параллельной ветви | Создаёт точку ветвления для логики ИЛИ |
| Ветка ↓ | Конец параллельной ветви | Объединяет ветви обратно |

### Типы привязки элементов

- **Ручное управление** — переключение кликом по элементу
- **Кнопка игры** — связь с кнопками ПУСК/СТОП на игровом поле
- **Датчик конвейера** — активация при прохождении заготовки
- **Выход (катушка)** — управление конвейером или другими устройствами

### Игровая симуляция

1. Нажмите **"▶ Запуск"** для старта симуляции
2. Используйте кнопки **ПУСК/СТОП** на игровом поле
3. Настройте LD-схему для автоматического управления:
   - Подключите кнопку ПУСК к НО контакту
   - Подключите кнопку СТОП к НЗ контакту (для безопасности)
   - Подключите катушку к выходу "Конвейер"

### Пример базовой схемы "Пуск-Стоп"

```
+24V --[НО Пуск]--[/НЗ Стоп]--(Конвейер)-- GND
                    |
                    |--[НО Конвейер]--|  (самоподхват)
```

**Логика работы**:
1. Нажатие ПУСК замыкает цепь → конвейер включается
2. Параллельный контакт Конвейер создаёт самоподхват
3. Нажатие СТОП размыкает цепь → конвейер выключается
